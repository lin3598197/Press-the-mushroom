# Press The Mushroom

一個只有巨大棕色蘑菇按鈕的極簡頁面。上方保留標題與計數器，蘑菇按鈕維持正方形並盡可能填滿視窗剩餘空間。

## 功能
- 點擊或鍵盤 Enter / Space 可遞增計數
- 響應式：蘑菇維持正方形，最大寬度 90vmin（或 900px）
- 簡單按壓/彈起動畫

## 結構
- `index.html` 主頁面
- `styles.css` 版面與蘑菇造型
- `script.js` 計數邏輯與動畫

## 使用
直接以瀏覽器開啟 `index.html`。
